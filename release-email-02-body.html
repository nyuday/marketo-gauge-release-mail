WELCOME

This add-on provides analytics at both the pipeline and global levels. You can get insights into pipeline runs, pipelines with highest wait time, and agents with the highest utilization. It is part of GoCD’s enterprise offering, which includes more enterprise add-ons and professional support. Learn more about the GoCD Enterprise Analytics Plugin.

Deprecations

  • As of release 18.2 the following unversioned APIs have been deprecated and will be removed in June 2018 - Pipeline pause, unpause, unlock and schedule APIs . These APIs have been replaced with a versioned API and users are encouraged to use these instead.
  • The pipeline config API version 3 and version 4 have been deprecated and will be removed in June 2018. Version 5 of the pipeline config API has been introduced to add the new pipeline unlock behavior attribute.
