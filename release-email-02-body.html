Security Update

We have released critical security fixes to prevent CSRF, XSS and Session hijacking exploits. We highly recommend users to upgrade to keep your GoCD server and agent secure.

Thanks 4cad for reporting these issues.

Found a critical security issue in GoCD that needs fixing? Please report it to https://hackerone.com/gocd

Feature Improvements

The in-built password file authentication plugin now supports bcrypt. It is recommended that GoCD server administrators migrate SHA1 hashed passwords, in the GoCD password file, to bcrypt.


We have made some improvements to existing features. We added an option to skip agent startup on windows, improve timestamp rendering (you can now see timestamps in your local timezone on key pages) and added a "Rerun Failed" button to rerun failed jobs.


We have also made some API enhancements and bug fixes for GoCD. A comprehensive list of changes can be found here.

LDAP/AD Authorization Commercial Plugin

We have a new LDAP/AD plugin which uses the Authorization Plugin Endpoint. The commercial version of the plugin allows you to grant access to your teams by leveraging your organizations' predefined roles within LDAP/AD. To know more, get in touch.

Breaking Changes

Version 1 of the Environment Config API was deprecated in 17.3. Version 2 of the API is available, and users are encouraged to use it.

Deprecations

These are a list of features that have been deprecated on GoCD along with the timeframe of when they will be removed completely.

  • Support for in-built password file and LDAP/AD authentication have been deprecated as of 17.5.0. Your configuration will automatically be updated to use the newly bundled file and ldap plugins.
  • Going forward, all the existing functionality provided by the Authentication Plugin will be handled by the new Authorization Plugin endpoints. Plugin developers are encouraged to migrate their plugins to use the new Authorization Plugins. The Authentication plugin has been deprecated as of 17.5.0 and we'd stop supporting it in 18.1 (scheduled to be release in January 2018).
