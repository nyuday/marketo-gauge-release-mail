New Dashboard out of Beta

After the release of the beta version of the new GoCD Dashboard in 18.2, we've made fixes and performance improvements based on feedback from our users. The GoCD dashboard improves performance for large GoCD instances with thousands of pipelines. In this release, the improved dashboard will be available as the default option to all GoCD users.



Enhancements to GoCD's Plugin Architecture

This release has improvements to our plugin architecture.


GoCD allows plugins to add their own Error and Warning messages which shows up in GoCD application.


Plugins can now implement multiple extensions. This will allow plugin developers to create a single plugin to support multiple extensions which complement each other.


Introduction of Version 3 of Notification Extension which allows adding of a new message to notify agent state changes.

Docker Images

A new Docker image for GoCD Agent is available for Debian 9.


  • We've also made other fixes and improvements in GoCD 18.3. A more comprehensive list of changes can be found in our release notes.

Deprecations

  • As of release 18.2 the following unversioned APIs have been deprecated and will be removed in June 2018 - Pipeline pause, unpause, unlock and schedule APIs . These APIs have been replaced with a versioned API and users are encouraged to use these instead.
  • The pipeline config API version 3 and version 4 have been deprecated and will be removed in June 2018. Version 5 of the pipeline config API has been introduced to add the new pipeline unlock behavior attribute.
  • The Elastic Agent API version 1 (and a development verion 2) has been deprecated. This will be removed in June 2018. Plugin developers should use version 3 of the Elastic Agent plugin API to allow adding support for agent status report and plugin status report. The Docker, Docker Swarm, and ECS plugins have been migrated to the new versions, and users should upgrade their plugins to the latest version to see these new features.
  • The Plugin Info API v3 has been deprecated as part of 18.3. This version of the API will be removed in July 2018. Version 4 of the API is available, and users are encouraged to use it.
  • The Agent Docker Image for Debian 7 is deprecated as part of 18.3 and will be removed in May 2018. Users should upgrade to a newer version of the Debian image.
