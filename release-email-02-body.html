Critical Fix

After the 17.9 release, we made an announcement asking users to hold off upgrading their GoCD instance. We found an issue (#3834) affecting a small set of users. This release provides a fix for the same.

OTHER CHANGES

We've fixed the issue where installed plugins were not showing up after the upgrade to 17.9. We've also made a few changes to the GoCD Server and the GoCD Agent Docker images.


A more comprehensive list of changes can be found in our release notes.
